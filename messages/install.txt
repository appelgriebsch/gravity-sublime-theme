How to enable:

To activate the theme for Sublime Text, add or replace your current theme settings with the code below.

Gravity Theme:

    "theme": "Gravity.sublime-theme",
    "color_scheme": "Packages/Theme - Gravity/Monokai Gravity.sublime-color-scheme"

Gravity Theme:

    "theme": "Gravity.sublime-theme",
    "color_scheme": "Packages/Theme - Gravity/One Dark Gravity.sublime-color-scheme"

Reverse Gravity Theme:

    "theme": "Reverse Gravity.sublime-theme",


macOS accent colors (use only one). Gray is default:

    "gravity_highlight_color_blue": true
    "gravity_highlight_color_purple": true
    "gravity_highlight_color_pink": true
    "gravity_highlight_color_red": true
    "gravity_highlight_color_orange": true
    "gravity_highlight_color_yellow": true
    "gravity_highlight_color_green": true
    "gravity_selected_tab_bold": true


Tabs: increase tab height:

    "gravity_tab_height_tall": true


Tabs: decrease tab height:

    "gravity_tab_height_short": true


Tabs: decrease tab font size:

    "gravity_tab_font_small": true


Tabs: bold text label for select tab:

    "gravity_selected_tab_bold": true


Blend macOS title bar with theme:

    "gravity_title_bar": true


Blend title bar with sidebar header:

    "gravity_sidebar_header": true
